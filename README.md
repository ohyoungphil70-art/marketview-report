## 🔍 리포트 활용하기

### 최신 리포트 확인

1. **[INDEX.md](INDEX.md)** 열기
2. 원하는 날짜 클릭
3. **[📄 보기]** 링크로 이메일 형식 확인

### 데이터 분석

1. **[data/archive/](data/archive/)** 폴더 열기
2. 특정 날짜의 JSON 파일 다운로드
3. Python/Excel에서 분석

### 월간 분석

1. **[reports/](reports/)** 폴더 열기
2. `YYYY-MM.md` 파일로 월간 요약 확인
